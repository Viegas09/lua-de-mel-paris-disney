⚡ 5 PASSOS RÁPIDOS:

PASSO 1: Criar Conta GitHub
  → Abra https://github.com
  → Clique "Sign up"
  → Preencha email, senha, username
  → Confirme no email
  ⏱️ 2 minutos

PASSO 2: Baixar Git
  → Windows: https://git-scm.com → Download for Windows
  → Mac: brew install git
  → Linux: sudo apt install git
  ⏱️ 3 minutos

PASSO 3: Criar Repositório no GitHub
  → Vá em https://github.com
  → Clique "+" → "New repository"
  → Nome: lua-de-mel-paris-disney
  → Clique "Create repository"
  ⏱️ 1 minuto

PASSO 4: Enviar Arquivos para GitHub
  → Leia PASSO_A_PASSO_GITHUB.md
  → Execute os 6 comandos Git
  → Pronto!
  ⏱️ 5 minutos

PASSO 5: Ativar GitHub Pages
  → No seu repositório GitHub
  → Settings → Pages
  → Source: main branch
  → Salve
  ⏱️ 1 minuto
