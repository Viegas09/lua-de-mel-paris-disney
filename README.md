# 🇫🇷 Lua de Mel - Paris & Disney 💕

App completo para organizar sua lua de mel em Paris e Disneyland (19-29 de Outubro 2026).

## 📋 O que inclui

✅ **Roteiro Completo** - 11 dias detalhados com horários EXATOS dos voos  
✅ **Voos Confirmados** - LATAM LA 8068 e LA 8067 com dados reais  
✅ **Hospedagens** - Montévrain (Disney) e Le Marais (Paris) com telefones  
✅ **Atrações** - Links para reservar Disney, Torre Eiffel, Louvre, Orsay, Versalhes  
✅ **Conversor EUR ↔ BRL** - Para saber quanto custa em reais  
✅ **Orçamento** - Rastreador de gastos com gráfico visual  
✅ **Notas Pessoais** - Para adicionar lembretes durante a viagem  
✅ **Modo Escuro** - Para conforto visual  
✅ **100% Offline** - Funciona sem internet  
✅ **Tudo Salvo Automaticamente** - No seu navegador
